# 余音 v0.2 用户使用文档

## 简介

余音是一款面向竹笛爱好者的 Web 音乐播放与学习工具。支持曲库管理、沉浸式音乐欣赏、简谱编辑、跟吹练习、音频分析等功能。

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/readant/Yuyin.git
cd Yuyin

# 创建虚拟环境
python -m venv venv
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt

# 启动
python launcher.py
```

浏览器自动打开 `http://localhost:5000`。

### 更新

```bash
git pull origin main
pip install -r requirements.txt
python launcher.py
```

## 页面功能

### 1. 曲库 `/`

管理你的音乐收藏。

**操作步骤：**
1. 点击「扫描文件夹」按钮
2. 输入音乐文件夹路径，点确定
3. 系统自动扫描 MP3/WAV/FLAC/OGG/M4A 文件
4. 在搜索框可搜索曲目
5. 点击「播放」进入播放器

**提示：** 扫描的目录会自动保存，重启后无需重新扫描。

---

### 2. 播放器 `/player`

沉浸式音乐欣赏界面。

**基础操作：**
- `|◀` / `▶|`：上一首 / 下一首
- `▶`：播放 / 暂停
- 进度条拖动：跳转到任意位置
- 音量：调整播放音量

**高级功能：**

| 按钮 | 功能 |
|------|------|
| 倍速 | 切换播放速度：0.5x / 0.75x / 1x / 1.25x / 1.5x / 2x |
| A-B | 循环播放选段：第一次点击设A点，第二次设B点，第三次取消 |
| 顺序 | 切换播放模式：顺序 / 随机 / 单曲循环 |
| 列表 | 打开播放列表侧边栏，点击曲目切换 |
| 歌词 | 加载LRC歌词文件或粘贴歌词文本，加载后在唱片位置显示歌词，随播放进度同步高亮 |

**歌词加载方式：**
1. 点击「歌词」按钮
2. 粘贴 LRC 歌词文本 → 点「加载」，或
3. 点「选择文件」选择 .lrc 文件
4. 歌词自动显示在唱片位置，当前行放大高亮
5. 再点「歌词」可关闭歌词恢复唱片显示

---

### 3. 简谱编辑器 `/editor`

创建和编辑竹笛洞洞谱。

**文本输入模式：**
1. 在文本框中输入简谱（空格分隔）：`1 2 3 | 4 5 6`
2. 点击「导入到洞洞谱」

**可视化编辑模式：**
- 使用指法键盘（低音/中音/高音按钮）点击输入音符
- 当前音符有朱砂色边框高亮
- 支持减时线（+/-按钮）、附点编辑
- 复制、粘贴、删除操作
- 延时线（-）、小节线（|）、间隔

**保存与导出：**
- 点击「保存简谱」存入数据库
- 「导出 JSON」下载为文件备份
- 「导入 JSON」恢复已保存的简谱
- 「导出 PDF」打印为乐谱

---

### 4. 练习室 `/practice`

竹笛跟吹练习和测试。

**两种模式：**

| 模式 | 功能 |
|------|------|
| **跟吹** | 不需要麦克风，纯视觉引导。节拍器按BPM推进，洞洞谱逐音高亮，指法图实时显示 |
| **测试** | 需要麦克风。在跟吹基础上加入实时音高检测和录音比对 |

**操作步骤：**
1. 从下拉菜单选择简谱
2. 设置 BPM（可用 -/+ 按钮微调）
3. 选择「跟吹」或「测试」模式
4. 点击「▶ 开始」
5. 跟随节拍器和洞洞谱吹奏
6. 点击「■ 停止」查看练习结果

**测试结果：**
- 正确/错误计数
- 准确率百分比
- 练习时长
- 逐音符比对（绿色匹配 / 红色不匹配）

---

### 5. 音频工具 `/tools`

专业音频分析和处理仪表盘。

**支持的操作：**

| 功能 | 说明 |
|------|------|
| 上传音频 | 支持拖拽或选择文件 |
| 录音 | 使用麦克风录制音频 |
| 波形图 | 可视化音频波形，带时间刻度 |
| 播放控制 | 内置播放器，播放时游标跟随 |
| 频谱分析 | 频率能量分布图 |
| 裁剪 | 指定起止时间（秒），裁剪音频片段 |
| 变调 | 调整音高（半音为单位） |
| 归一化 | 统一音量峰值 |
| 生成简谱 | AI 识别音频音高，自动生成简谱文本 |
| 对比练习 | 加载参考音频 + 录制自己的演奏 → 音高序列比对 |

---

### 6. 歌词编辑 `/lyrics`

编辑和管理 LRC 歌词文件。

**操作步骤：**
1. 粘贴 LRC 歌词文本或选择 .lrc 文件
2. 点击「加载歌词」解析
3. 右侧预览区显示歌词行列表
4. 点击某行可编辑时间和文本
5. 「导出 LRC」保存为文件

**LRC 格式说明：**
```
[00:15.000]时光穿不断 流转在从前
[00:22.000]刻骨的变迁 不是遥远
```
每行格式为：`[分:秒.毫秒]歌词内容`

---

### 7. 设置 `/settings`

个性化你的学习体验。

| 设置项 | 说明 |
|--------|------|
| 默认调性 | D/G/C/F/E 调 |
| 默认指法 | 筒音作5/1/2 |
| 默认 BPM | 练习时的默认速度 |
| 主题模式 | 浅色 / 深色 / 护眼（点击色块预览） |
| 字体大小 | 14/16/18/20px |
| 内容宽度 | 窄/标准/宽/全宽 |
| 侧边栏 | 收窄/中等/标准 |
| 动画效果 | 开启/关闭 |
| 背景图片 | 上传自定义背景，选择或删除 |
| 数据管理 | 清空曲库 / 清空简谱 / 查看练习统计 |

## 常见问题

**Q: 扫描文件夹后没有曲目？**
A: 确保文件夹内包含支持的格式（MP3/WAV/FLAC/OGG/M4A）。

**Q: 歌词不同步？**
A: LRC 歌词的时间戳需要与音频文件的实际播放时间匹配。可在歌词编辑页面调整时间。

**Q: 测试模式探测不到音高？**
A: 确保浏览器已授权麦克风权限，保持环境安静，竹笛对准麦克风吹奏。

**Q: 简谱生成不准确？**
A: AI 音高识别依赖音频质量，纯器乐录音效果最佳。可在编辑器中手动调整。

**Q: 如何备份数据？**
A: 简谱数据存储在 `data/zhudi.db`，练习记录在 `data/practice_records.json`，听歌记录在 `data/listen_records.json`。
