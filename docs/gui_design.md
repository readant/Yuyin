# 余音 GUI 设计方案

## 设计目标

从"静态皮肤"升级为"会呼吸的国风画面"，打造年轻人喜爱的沉浸式音乐体验。

---

## 一、视觉沉浸

### 1.1 墨韵粒子系统

**效果设计**：
- 播放时根据音乐节奏触发粒子
- 鼓点落下：墨点在宣纸上晕开
- 笛声响起：柳叶/花瓣缓缓飘落
- 戏曲副歌：细碎金粉飘散
- 暂停时粒子慢慢消散，呼应"余音绕梁"

**PyQt6 落地方式**：
- QGraphicsScene + QGraphicsPixmapItem 封装轻量粒子类
- librosa 实时提取音频节拍、能量峰值控制粒子
- 进阶：QOpenGLWidget + GLSL 着色器实现水墨扩散

**曲风自动切换**：
- 古琴曲：银杏叶
- 仙侠风：萤火
- 戏曲：金粉

### 1.2 动态氛围感背景

**方案一：云纹流动**
- 正弦函数驱动两层半透明云纹 SVG 缓慢错位移动
- QSvgRenderer 渲染，无额外依赖

**方案二：烛影摇曳**
- 界面角落动态烛火
- 明暗缓慢呼吸变化
- 音量调低时烛火变暗，静音时熄灭

**方案三：窗影婆娑**
- 模拟月光透过窗棂投在宣纸上的影子
- 缓慢小幅晃动，深夜书房感觉

### 1.3 墨迹落笔式歌词动效

**效果**：
- 唱到哪个字，哪个字从淡墨团凝聚成清晰笔锋
- 竖排模式下像古人提笔写就
- 高潮句周围泛起淡淡墨色光晕

**实现**：
- QGraphicsTextItem + QGraphicsBlurEffect
- 模糊半径从大到小过渡

### 1.4 毛玻璃宣窗效果

**效果**：
- 侧边栏、设置弹窗弹出时
- 背景半透明 + 轻微模糊 + 宣纸纹理
- 像隔着书房纱窗看外面

**实现**：
- QGraphicsBlurEffect + 半透明宣纸蒙版
- 裁剪成圆角矩形或扇面形

## 二、交互巧思

### 2.1 拟物化国风控件

**播放/暂停按钮**：
- 做成古代"磬"的形状
- 点击播放：伴随磬音，按钮轻微下沉
- 暂停：收声的余韵感

**进度条**：
- 做成卷轴样式
- 两端木质卷轴轴头
- 拖动像展开/收起画卷
- 结尾盖"终"字朱砂印章

**音量滑块**：
- 做成"拨弦"样式
- 拖动像拨动琴弦
- 每一格对应古筝泛音
- 最大时琴弦"绷紧"视觉反馈

### 2.2 国风转场动效

**卷帘转场**：
- 竹帘从上方缓缓卷起
- 适合从上往下展开的页面

**墨晕转场**：
- 墨点从中心晕开覆盖全屏
- 再慢慢收束消失
- 适配所有场景

**翻页转场**：
- 像翻阅线装书
- 页面从右往左带透视角度翻过去
- 适合"藏曲阁"歌单页面

### 2.3 悬停微反馈

**歌单列表项**：
- 悬停时左侧浮现梅花/兰草标记
- 像古籍里的朱砂批注

**按钮悬停**：
- 颜色从浅红加深到朱砂红
- 轻微放大一点点
- 像印章慢慢盖实

**歌词行悬停**：
- 旁边出现"注"字图标
- 点击可查看典故/生僻字注音

### 2.4 桌面悬浮窗

**最小化效果**：
- 变成印章/扇面悬浮在桌面角落
- 只显示歌名和极简进度条
- 支持切歌、暂停

**进阶：竹简式迷你歌词**：
- 竖排从右往左缓慢滚动
- 像桌面的一行小诗

## 三、听觉沉浸

### 3.1 全国风操作音效

**音效映射**：
- 点击按钮：短促木鱼声
- 切换歌单/页面：轻拨古筝泛音
- 播放启动：清脆小磬声
- 拖动进度条：细碎竹简摩擦声

**实现**：
- QSoundEffect 播放短 wav 音频
- 开源民乐采样包

### 3.2 余音式淡入淡出

**播放启动**：
- 音量从 0 缓慢淡入
- 像琴声从远处飘来

**点击暂停**：
- 音量缓缓淡出
- 像余音慢慢消散

**切歌**：
- 前一曲淡出、后一曲淡入
- 衔接丝滑

### 3.3 国风白噪音叠加

**雅境模式**：
- 听雨轩：小雨淅沥 + 远处雷声
- 松风亭：山风穿过松林
- 煮茶阁：炭火噼啪 + 沸水轻声

**实现**：
- pydub 或 vlc 做音频混流
- 无缝循环环境音素材
- 单独控制音量

## 四、彩蛋式设计

### 4.1 一键生成国风歌词海报

**功能**：
- 右键歌词一键生成
- 宣纸底纹 + 书法字体 + 落款印章
- 自动适配歌曲意境
- 可保存分享到朋友圈、小红书

**模板**：
- 水墨风格
- 敦煌风格
- 工笔风格

**实现**：
- Pillow 库做图片合成

### 4.2 音律签文

**触发条件**：
- 听完一首完整的歌
- 右下角弹出"今日音律签"

**签文内容**：
- "今日曲风：清欢 宜练字，宜静心"
- 一句对应意境的古诗词

**实现**：
- 低成本：本地预置几百条签文随机匹配
- 进阶：接入本地 Ollama 大模型

### 4.3 节气自动换肤

**触发条件**：
- 二十四节气/传统节日当天
- 打开软件自动切换对应主题

**示例**：
- 清明：淡柳色主调 + 柳絮粒子
- 中秋：月白色界面 + 桂花飘落
- 大雪：素白底色 + 细雪粒子

### 4.4 古籍式文案体系

**文案映射**：
- 我的歌单 -> 藏曲阁
- 最近播放 -> 近日所闻
- 收藏 -> 倾心之音
- 设置 -> 雅室陈设
- 搜索 -> 寻曲

## 五、性能与规范

### 5.1 性能优先

- 所有动效、粒子放在单独线程
- 粒子数量控制在 50 个以内
- 背景动效帧率锁 30 帧
- 不能卡顿、不能影响音乐播放

### 5.2 克制留白

- 国风高级感靠留白
- 主界面只保留 1-2 个动态元素
- 不要同时放粒子、动态背景、发光歌词

### 5.3 可关闭

- 所有特效、音效都给开关
- 允许用户切换"极简模式"
- 照顾性能差的电脑和喜欢安静的用户

### 5.4 版权合规

- 图标、音效、纹理都用开源可商用素材
- 避免侵权风险
